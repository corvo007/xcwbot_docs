### 存轴器
<b>相关服务:存轴器</b>
::: danger 警告
初次使用时需要设置数据库名称，请设置一个独特的名字，避免被其他人通过`切换数据库`命令读取。
:::
::: tip 提示
1.`查找轴 A/B 1/2/3/4/5`指令只会列出指定Boss的轴列表，类似:  
<b>编号|boss|伤害|备注|赞同数  
T01|A1|1|test|0赞  
T02|A6|2|test|0赞  </b>  
须再使用`查找轴 编号(如上所示)`查看每个轴的详情。
:::

指令|说明
:--|:--|:--
录入轴 A/B 1/2/3/4/5 伤害 说明 轴| 
查找轴 A/B 1/2/3/4/5 |
查找轴 编号 |
更新轴 编号 伤害 说明 轴 |
删除轴 编号 |
赞同轴 编号 |赞同数，非管理员只能填1/-1
查看轴库名|
切换轴库 新轴库名 |

::: tip 提示
如需清空轴库，只需切换到一个新的轴库即可，无需操作原来的轴库。
:::
